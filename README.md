# Mizu
Mizu is the forever-free, fully open source, completely cross-platform successor to Anki. Mizu is intended to be a feature-complete replacement of the beloved Anki application.

## Features
- Spaced repetition flashcards with built-in FSRS (Free Spaced Repition Scheduler)
- Unlimited decks and sub-decks
- Card templates that support Basic cards, Cloze deletion cards, and media (images, video, audio)
- Create custom card templates
- Card browser for bulk editing, tagging, etc.
- Built-in, free, local text-to-speech (TTS) generation
- Robust, thoroughly documented plugin system available for both desktop and mobile
- Available for desktop (Windows. macOS, Linux) and mobile (iPhone, Android) with **100% feature parity, including plugins**
- Sync/backup freedom: local-first, folder-based collection that allows you to use your own backup or or sync approach: keep local, use cloud drive sync (e.g., Google Drive), or use Mizu's cloud sync. This is similar to Obsidian.
- GPL 3 license
